# 🌞 D.D. Solution

ระบบจัดการธุรกิจโซล่าเซลล์ — ติดตามงาน, สต๊อก, ผู้ลงทุน, ลูกค้า, และบัญชี

## ✨ ฟีเจอร์

- 🔐 **Login System** - 3 PIN สำหรับ พ่อ/โฟน/อาม
- 📊 **Dashboard** - ภาพรวมธุรกิจทั้งหมด
- 💼 **จัดการงาน** - ติดตามทุกงานติดตั้ง พร้อมต้นทุน/กำไร
- 📦 **สต๊อกอัจฉริยะ** - คิดสายไฟเป็นเมตร, เบิกเข้างานอัตโนมัติ
- 👥 **ผู้ลงทุน** - คำนวณเงินลงทุน + ปันผล
- 👤 **ลูกค้า** - ที่อยู่, เบอร์โทร, Google Maps
- 💰 **บัญชี** - รายรับ-รายจ่าย พร้อม Export Excel
- 📝 **Activity Log** - ประวัติการแก้ไขทุกอย่าง
- 🌐 **Online Sync** - ข้อมูล sync ทุกเครื่อง (Firebase)
- 📱 **PWA Support** - ติดตั้งเป็นแอปบนมือถือได้

## 🚀 ติดตั้ง

ดูคู่มือเต็มที่ [SETUP.md](./SETUP.md)

## 🛠️ Tech Stack

- React 18 (via CDN)
- Tailwind CSS (via CDN)
- Firebase (Firestore + Auth)
- SheetJS (Excel export)
- PWA (Offline support)

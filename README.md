# 🐢 D.D. Solution

ระบบจัดการธุรกิจโซล่าเซลล์ — ติดตามงาน, สต๊อก, ผู้ลงทุน, ลูกค้า, และบัญชี

## ✨ ฟีเจอร์

- 🔐 **Login System** - 3 PIN สำหรับ พ่อ/โฟน/อาม
- 📊 **Dashboard** - ภาพรวมธุรกิจทั้งหมด
- 💼 **จัดการงาน** - ติดตามทุกงานติดตั้ง พร้อมต้นทุน/กำไร
- 📦 **สต๊อกอัจฉริยะ** - คิดสายไฟเป็นเมตร, เบิกเข้างานอัตโนมัติ
- 👥 **ผู้ลงทุน** - คำนวณเงินลงทุน + ปันผล
- 👤 **ลูกค้า** - ที่อยู่, เบอร์โทร, Google Maps
- 💰 **บัญชี** - รายรับ-รายจ่าย พร้อม Export Excel แบบสวยงาม
- 📝 **Activity Log** - ประวัติการแก้ไขทุกอย่าง
- 🌐 **Online Sync** - ข้อมูล sync ทุกเครื่อง (Firebase)
- 📱 **PWA** - ติดตั้งเป็นแอปบนมือถือได้ ใช้งาน offline

## 🚀 ติดตั้ง

ดูคู่มือเต็มที่ [SETUP.md](./SETUP.md)

## 📱 ติดตั้งเป็นแอปบนมือถือ

### iPhone (Safari):
1. เปิดเว็บใน **Safari**
2. กดปุ่ม **Share** (สี่เหลี่ยม + ลูกศรขึ้น)
3. เลือก **"Add to Home Screen"**

### Android (Chrome):
1. เปิดเว็บใน **Chrome**
2. กดเมนู ⋮ มุมขวาบน
3. เลือก **"Install app"** หรือ **"Add to Home screen"**

## 🛠️ Tech Stack

- React 18 (via CDN)
- Tailwind CSS (via CDN)
- Firebase (Firestore + Auth)
- xlsx-js-style (Excel export with styling)
- PWA (Service Worker + Manifest)
